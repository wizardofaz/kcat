Frequency Calibration Dialog

Select the frequency for the frequency calibration measurement.  The MHz is selected by the outside arrow buttons and tenths of MHz with the inner buttons.  The range for frequency measurement is 5.0 to 15.0 MHz.

After selecting the frequency press the Calibrate button.  The transceiver will be set to receive on the selected frequency using CW in 1000 Hz bandwidth.  You should hear the remote signal at your usual CW sidetone frequency.  If you have the sidetone set to 700 Hz then you will hear a 700 Hz tone.  The transceiver will now make a series of signal level measurements to determine whether there is sufficient strength upon which to make the measurement.  You will be notified of its viability in accordance with:

< 0.1 uV Too small No measurement taken
0.1 to 1.0 uV Marginal Option to calibrate given
> 1.0 uV Sufficient Measurement made

If the signal reports as too low or of marginal quality you may select a different frequency and press calibrate again to test that frequencies viability.

Upon successful calibration a notification message box is displayed.  You may then return to the normal receiver operation by pressing Finished.

If you are also using fldigi you can fine tune the calibration using it's Frequency Analysis mode.

  1. Select the Analysis mode in fldigi
  2. Set the waterfall to the x4 scale
  3. Set kcat for USB, 2700 Hz bandwidth
  4. Set the frequency to the WWV frequency less 1000 Hz, ie: 9999.000 kHz
  5. Adjust the Vfo Adj control to zero beat the 1000 Hz detected WWV carrier.
    As you adjust the control the carrier will move with respect to the ideal.
